Human and Operational Performance Insight Tool

Overview

This application is a decision support system designed for industrial and technical environments. It integrates workforce analytics, operational performance metrics, and continuous improvement methodologies into a unified interface.

The tool bridges the gap between raw operational data and organizational decision-making, providing leaders and technical managers with a clear view of efficiency, skill gaps, and improvement opportunities.

Key Features

Executive Dashboard
Provides a high-level view of critical KPIs including efficiency, incidents, and total workforce health.

Workforce Analytics
Offers a deep dive into the organization by role, seniority, and sector. Includes individual employee profiles to track performance history.

Operational Performance
Tracks production volumes, efficiency rates vs. standard times, and analyzes quality costs by differentiating rework from scrap.

Skill Matrix & Training
Visualizes team versatility and automatically detects skill gaps to prioritize training needs.

Continuous Improvement Engine
Uses a rule-based recommendation system to suggest specific methodologies (Kaizen, 5S, Lean) based on detected operational risks.

Technical Context

Built using Python and Streamlit, this project demonstrates a modular architecture with a focus on clean data handling using Pandas. It includes a simulated data engine to generate realistic industrial scenarios for demonstration purposes.

Installation

Clone the repository and install the required dependencies found in requirements.txt. Run the application using the streamlit run command pointing to app.py.
